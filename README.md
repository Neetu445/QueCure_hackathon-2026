**Overview**
Queue Cure is a real-time clinic queue management system designed to replace manual
token slips and reduce patient waiting confusion. It provides a digital platform for
receptionists to manage patients and for patients to track their waiting status.


**Features**
- Receptionist dashboard to add patients and manage tokens
- Call next patient functionality
- Real-time queue updates using Socket.IO
- Patient waiting room showing current token and estimated wait time
- Dynamic wait time calculation based on queue length and consultation time

**Technology Stack**
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-time Communication: Socket.IO

**Problem Solved**
Many clinics still use manual token systems causing long waiting times and poor visibility.
Queue Cure provides transparency and improves clinic workflow.
Future Scope
- Online appointment booking
- Doctor dashboard
- Analytics for clinic performance
- Multi-clinic support
